# Plotline Marketing — UI Kit

The marketing landing page for Plotline. Open `index.html`.

## Sections
- **Nav** — brand lockup, links, sign-in / start-trial.
- **Hero** — headline, lede, dual CTA, and a product visual: a paper plan sheet on the
  blueprint-grid canvas with live area/linear/count markups and value callouts + a mini tool
  rail. This is the signature brand image.
- **Logo strip** — social proof.
- **Features** — six cards, each with a takeoff-type-colored icon tile.
- **How it works** — three numbered steps.
- **Stats band** — dark inverse surface with mono figures (brand-green accents).
- **CTA** + **Footer**.

## Built from the design system
Uses `Button` (and `Badge`) from the bundle; all color, type, spacing, shadow, and the
blueprint-grid / paper motifs come from `styles.css`. Icons via Lucide + `assets/icons.js`.
Voice follows the README's content fundamentals: sentence case, trade-literate, mono for any
number ("3× faster," "12k+ sheets," "4,820 sq ft").

Cosmetic recreation — links are non-functional.
